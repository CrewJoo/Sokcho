# E2E 테스트 실행

// turbo-all

1. Playwright E2E 테스트를 실행합니다:
```bash
cd e:\Demo\007_PREP_5Dsay && npx playwright test
```

2. 테스트 결과를 확인합니다. 실패한 테스트가 있으면 원인을 분석합니다.

3. (선택) 실패한 테스트의 리포트를 확인합니다:
```bash
cd e:\Demo\007_PREP_5Dsay && npx playwright show-report
```
